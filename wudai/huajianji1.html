不知所以 (bù zhī suǒ yǐ)

Abbreviation: bzsy

Explanation: 以原因。不明白为什么是这样。指不知道原因。

Derivation: 唐·张彦远《法书要录·购兰亭序》遽见追乎，不知所以。”

Example: 咪咪忽然拿起一只锅盖，用一只骨筷乒乒乓乓敲起来，敲得他～，惊惶失措，晕头转向。★王安忆《流逝》

不知所云 (bù zhī suǒ yún)

Abbreviation: bzsy

Explanation: 云说。不知道说得是些么。形容说话内容混乱，无法理解。

Derivation: 三国·蜀·诸葛亮《出师表》临表涕泣，不知所云。”

Example: 这篇文章逻辑性太差，使人看了～。

不知所终 (bù zhī suǒ zhōng)

Abbreviation: bzsz

Explanation: 终最后，结局。不知道结局或下落。

Derivation: 《庄子·田子方》日夜无隙，而不知其所终。”《国语·越语下》遂乘扁舟，以浮于五湖，莫知其所终极。”

Example: 余幼在湖口，有一僧持显者书，周行各郡县……后～。★清·采蘅子《虫鸣漫录》

不知疼痒 (bù zhī téng yǎng)

Abbreviation: bzty

Explanation: 是疼是痒都不知道。比喻不体贴，不关心。

Derivation: 明·施耐庵《水浒传》第十六回便是留守相公自来监押時，也容我们说一句。你好不知疼痒，只顾逞办！”

Example: 无

不知天高地厚 (bù zhī tiān gāo dì hòu)

Abbreviation: bztgdh

Explanation: 不知道天有多高，地有多深。形容骄狂无知。

Derivation: 《庄子·让王》吾不知天之高也，地之下也。”

Example: 如今年过知非，想起幼年这些～的话来，真觉后悔。★清·文康《儿女英雄传》第三十四回

不知痛痒 (bù zhī tòng yǎng)

Abbreviation: bzty

Explanation: 是痛是痒都不知道。比喻不体贴，不关心。

Derivation: 《二刻拍案惊奇》卷十二而今的世界只管讲那道学，说正心诚意的，多是一班害了风痹病，不知痛痒之人。”

Example: 无

不知头脑 (bù zhī tóu nǎo)

Abbreviation: bztn

Explanation: 弄不清楚是怎么回事。犹不知所措。

Derivation: 明·凌濛初《初刻拍案惊奇》卷三十又恐怕因此惹恼了他，连自己身子立不勾，却又不见得李参军触恼他一些处，正是不知一个头脑。”

Example: 无

不知有汉，何论魏晋 (bù zhī yǒu hàn，hé lùn wèi jìn)

Abbreviation: bzyhlwj

Explanation: 不知道有汉朝，三国魏及晋朝就更不知道了。形容因长期脱离现实，对社会状况特别是新鲜事物一无所知。也形容知识贫乏，学问浅薄。

Derivation: 晋·陶渊明《桃花源记》先世避秦时乱，率妻子邑人，来此绝境，不复出焉，遂与外人间隔。问今是何世，乃不知不汉，何论魏晋。”

Example: 无

不知者不罪 (bù zhī zhě bù zuì)

Abbreviation: bzzbz

Explanation: 罪责备，怪罪。因事先不知道而有所冒犯，就不加怪罪。

Derivation: 清·钱彩《说岳全传》第六十三回起凤道‘牛兄何不早通姓名，使小弟多多得罪。勿怪，勿怪。’牛通道‘不知者不罪。’”

Example: 无

不知自爱 (bù zhī zì ài)

Abbreviation: bzza

Explanation: 不知道爱惜自己。

Derivation: 无

Example: 父母师长如此爱护他。他却～，自甘堕落，怎不令人痛惜！

不知自量 (bù zhī zì liàng)

Abbreviation: bzzl

Explanation: 形容没有正确估计自己的能力有多大。

Derivation: 无

Example: 我接下这项工程后，才发觉自己有些～。

不直一钱 (bù zhí yī qián)

Abbreviation: bzyq

Explanation: 鄙视之词，犹言毫无价值。比喻无能或品格卑下。

Derivation: 语出《史记·魏其武安侯列传》夫无所发怒，乃骂临汝侯曰‘生平毁程不识不直一钱，今日长者为寿，乃效女儿咕嗫耳语。”

Example: 无

不直一文 (bù zhí yī wén)

Abbreviation: bzyw

Explanation: 鄙视之词，犹言毫无价值。比喻无能或品格卑下。

Derivation: 宋·罗大经《鹤林玉露》卷十四士大夫若爱一文，不直一文。”

Example: 无

不值一驳 (bù zhí yī bó)

Abbreviation: bzyb

Explanation: 不值得批驳。

Derivation: 无

Example: 对于他所提有关修改方案的提议，完全没有事实根据，～。

不值一顾 (bù zhí yī gù)

Abbreviation: bzyg

Explanation: 顾看。不值得一看。形容瞧不起某事物。

Derivation: 明·方孝儒《送龚彦佐序》夫禄之以天下而系马千驷，常人思以其身易之而不可得，而伊尹不屑一顾视焉。”

Example: 相反地，被认为是软弱的表现，～。★毛泽东《中共中央毛泽东主席善于时局的声明》

不值一钱 (bù zhí yī qián)

Abbreviation: bzyq

Explanation: 比喻毫无价值。

Derivation: 《史记·魏其安侯列传》生平毁程不识不直一钱。”

Example: 此两公俱以直臣起家致大位，晚途遭诟，～，宪体至是扫地矣。★明·沈德符《万历野获编·御史大夫被论》

不值一哂 (bù zhí yī shěn)

Abbreviation: bzys

Explanation: 哂微笑。不值得一笑。比喻毫无价值。也表示对某种事物或行为的轻蔑和讥笑。

Derivation: 鲁迅《二心集·非革命的急进革命论者》凡大队的革命军，必须一切战士的意识都十分正确、分明，这才是真正的革命军，否则不值一哂。”

Example: 无

不值一笑 (bù zhí yī xiào)

Abbreviation: bzyx

Explanation: 不值得一笑。比喻毫无价值。也表示对某种事物或行为的轻蔑和讥笑。同不值一哂”。

Derivation: 鲁迅《朝花夕拾·死后》但同时想，我生存时说的什么批评不值一笑的话，大概是违心之论罢。”又《呐喊·阿Ｑ正传》阿Ｑ又很自尊，所有未庄的居民，全不在他眼神里，甚而至于对于两位‘文童’也有以为不值一笑的神情。...

Example: 无

不忮不求 (bù zhì bù qiú)

Abbreviation: bzbq

Explanation: 忮嫉妒；求贪求。指不妒忌，不贪得无厌。

Derivation: 《诗经·邶风·雄雉》百尔君子，不知德行。不忮不求，何用不臧。”

Example: 处理了这些事务，便是长篇大论的遗训”，教子孙～，克勤克俭。★高阳《玉座珠帘》下册

不治之症 (bù zhì zhī zhèng)

Abbreviation: bzzz

Explanation: 医治不好的病。也比喻无法挽救的祸患。

Derivation: 明·冯梦龙《醒世恒言》卷十太医诊了诊脉，说道‘……此乃不治之症。’”

Example: 假使她们真知道立国数千年的大中华民国”的国民，往往有自欺欺人的～，那可真是没有面子了。★鲁迅《二心集·以脚报国》
