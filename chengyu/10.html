饮谷栖丘 (yǐn gǔ qī qiū)

缩写: ygqq

解释: 犹言栖丘饮谷。指隐逸山林。

出自: 无

造句: 无

饮河满腹 (yǐn hé mǎn fù)

缩写: yhmf

解释: 比喻人应知足，贪多无益。

出自: 《庄子·逍遥游》鹪鹩巢于深林，不过一枝；偃鼠饮河，不过满腹。”

造句: 无

饮河鼹鼠 (yǐn hé yǎn shǔ)

缩写: yhys

解释: 比喻指所需求或所得极有限的人。

出自: 语出《庄子·逍遥游》偃鼠饮河，不过满腹。”明·李昌祺《剪灯余话·洞天花烛记》饮河鼹鼠愧盈腹，止鲁鶢鶋惭厚享。”

造句: 每一个人在人类智慧的长河旁边，都不过像一只饮河的鼹鼠。★秦牧《社稷坛抒情》

饮恨而终 (yǐn hèn ér zhōng)

缩写: yhez

解释: 饮恨心里怀着怨恨和冤屈。内心满怀着怨恨或冤屈死去。

出自: 唐·蒋防《霍小玉传》我为女子，薄命如斯。君是丈夫，负心如此。韶颜稚齿，饮恨而终。”

造句: 小女兴娘薄命，为思念郎君成病，于两月前～，已殡在郊外了。★明·凌濛初《初刻拍案惊奇》卷二十三

饮恨吞声 (yǐn hèn tūn shēng)

缩写: yhts

解释: 饮恨强忍怨恨；吞声哭泣而不敢出声。形容忍恨含悲，不敢表露。

出自: 南朝梁·江淹《恨赋》自古皆有死，莫不饮恨而吞声。”

造句: 无

饮灰洗胃 (yǐn huī xǐ wèi)

缩写: yhxw

解释: 灰古代以草木灰作洗涤剂。比喻悔过自新。

出自: 《南史·荀伯玉传》若许某自新，必吞刀刮肠，饮灰洗胃。”

造句: 无

饮露餐风 (yǐn lù cān fēng)

缩写: ylcf

解释: 喝的是露水，吃的是风。形容超凡脱俗的生活。亦作饮风餐露”。

出自: 无

造句: 无

饮马长江 (yǐn mǎ cháng jiāng)

缩写: ymcj

解释: 在长江边给战马喝水。指渡江南下进行征伐。

出自: 无

造句: 无

饮马投钱 (yìn mǎ tóu qián)

缩写: ymtq

解释: 比喻为人廉洁，不损公肥私。

出自: 唐·徐坚《初学记》卷六引《三辅决录》安陵清者有项仲仙，饮马渭水，每投三钱。”

造句: 无

饮马投钱 (yìn mǎ tóu qián)

缩写: ymtq

解释: 比喻为人廉洁，不损公肥私。

出自: 唐·徐坚《初学记》卷六引《三辅决录》安陵清者有项仲仙，饮马渭水，每投三钱。”

造句: 无

饮气吞声 (yǐn qì tūn shēng)

缩写: yqts

解释: 指不喘气，不做声。形容忧惧惶恐。

出自: 无

造句: 无

饮泣吞声 (yǐn qì tūn shēng)

缩写: yqts

解释: 眼泪只能往肚里流，不敢哭出声来。形容受压迫时，忍受痛苦，不敢公开表露。

出自: 明·施耐庵《水浒全传》第九十八回琼英知了这消息，如万箭攒心，日夜吞声饮泣，珠泪偷弹，思报父母之仇，时刻不忘。”

造句: 无

饮泉清节 (yǐn quán qīng jié)

缩写: yqqj

解释: 指廉正清白的节操。

出自: 《晋书·良吏传·吴隐之》载虏中，吴为广州刺史，经石门，有水名贪泉，饮者怀无厌之欲”，吴至泉所，酌而饮之，因赋诗‘古人云此水，一歃怀千金。试使夷齐饮，终当不易心。’”抵任后，吴清操逾厉”。

造句: 突兀高台累土成，人言暴客此分赢。～今寥落，可但梁山独擅名！★明·刘基《分赃台》诗

饮食男女 (yǐn shí nán nǚ)

缩写: ysnn

解释: 泛指人的本性。

出自: 《礼记·礼运》饮食男女，人之大欲存焉。”

造句: 无

饮食起居 (yǐn shí qǐ jū)

缩写: ysqj

解释: 指人的日常生活。

出自: 无

造句: 无

饮水辨源 (yǐn shuǐ biàn yuán)

缩写: ysby

解释: 比喻明白事物的本源。同饮水知源”。

出自: 无

造句: 无

饮水啜菽 (yǐn shuǐ chuò shū)

缩写: yscs

解释: 饿了吃豆羹，渴了喝清水。形容生活清苦。

出自: 《荀子·天论》君子啜菽饮水，非愚也，是节然也。”《礼记·檀弓下》孔子曰啜菽饮水尽其欢，斯之谓孝。”

造句: 无

出自: 无

造句: 无

饮水曲肱 (yǐn shuǐ qū gōng)

缩写: ysqg

解释: 形容清心寡欲、安贫乐道的生活。

出自: 《论语·述而》子曰‘饭疏食饮水，曲肱而枕之，乐亦在其中矣。不义而富且贵，于我如浮云。’”

造句: 无

缩写: ysss

解释: 形容生活清苦。同饮水啜菽”。
