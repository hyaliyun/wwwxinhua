明公正气 (míng gōng zhèng qì)

缩写: mgzq

解释: 犹言正式；公开；堂堂正正。

出自: 无

造句: 无

明公正义 (míng gōng zhèng yì)

缩写: mgzy

解释: 犹言正式；公开；堂堂正正。同明公正气”。

出自: 无

造句: 无

明鼓而攻之 (míng gǔ ér gōng zhī)

缩写: mgegz

解释: 宣布罪状，遣责或声讨。

出自: 《论语·先进》非吾徒也，小子鸣鼓而攻之可也。”

造句: 无

明光烁亮 (míng guāng shuò liàng)

缩写: mgsl

解释: 指光亮耀眼。

出自: 无

造句: 无

明光锃亮 (míng guāng zèng liàng)

缩写: mgzl

解释: 指光亮耀眼。同明光烁亮”。

出自: 无

造句: 无

明光铮亮 (míng guāng zhēng liàng)

缩写: mgzl

解释: 指光亮耀眼。同明光烁亮”。

出自: 无

造句: 无

明婚正配 (míng hūn zhèng pèi)

缩写: mhzp

解释: 指经过正式手续的婚姻，现指符合婚姻法规定的合法婚姻。

出自: 元·杨显之《潇湘雨》第四折你小姐元是我崔文远明婚正配许与侄儿崔通的。”

造句: 后来虽然做了亲，元不是～。★明·冯梦龙《二刻拍案惊奇》第11卷

明婚正娶 (míng hūn zhèng qǔ)

缩写: mhzq

解释: 指经过正式手续的婚姻，现指符合婚姻法规定的合法婚姻。

出自: 元·关汉卿《救风尘》第四折那里是明婚正娶，公然的伤风败俗。”

造句: 那曹家女子，或者说是～，又且结发在先。★清·李渔《凰求凤·作难》

明火持杖 (míng huǒ chí zhàng)

缩写: mhcz

解释: 形容公开抢劫或肆无忌惮地干坏事。同明火执仗”。

出自: 无

造句: 无

明火执械 (míng huǒ zhí xiè)

缩写: mhzx

解释: 形容公开抢劫或肆无忌惮地干坏事。同明火执仗”。

出自: 无

造句: 无

明火执仗 (míng huǒ zhí zhàng)

缩写: mhzz

解释: 明点明；执拿着；仗兵器。点着火把，拿着武器。形容公开抢劫或肆无忌惮地干坏事。

出自: 元·无名氏《盆儿鬼》第二折我在这瓦窑居住，做些本份生涯，何曾明火执仗 ，无非赤手求财。”

造句: 有一夜，无端被强盗～的抢了进来。★清·吴趼人《二十年目睹之怪现状》第29回

明火执杖 (míng huǒ zhí zhàng)

缩写: mhzz

解释: 点着火把，拿着武器。原指公开抢劫◇比喻公开地、毫不隐藏地干坏事。

出自: 元·无名氏《盆儿鬼》第二折我在这瓦窑居住，做些本份生涯，何曾明火执仗 ，无非赤手求财。”

造句: 并非～，怎么便算是强盗呢？★清·曹雪芹《红楼梦》第一百一十回

明见万里 (míng jiàn wàn lǐ)

缩写: mjwl

解释: 对于外界或远方的情况，知道得很清楚。也比喻人有预见。

出自: 《后汉书·窦融传》书妪至，河西咸惊，以为天子明见万里之外。”

造句: 无

明教不变 (míng jiào bù biàn)

缩写: mjbb

解释: 教令明确，让老百姓有所适从，便不会发生变乱。

出自: 《商君书·赏刑》夫明赏不费，明刑不戮，明教不变，而民知于民务，国无异俗。”

造句: 无

明镜高悬 (míng jìng gāo xuán)

缩写: mjgx

解释: 传说秦始皇有一面镜子，能照人心胆。比喻官员判案公正廉明。

出自: 《西京杂记》卷三有方镜，广四尺，高五尺九寸，表里有明，人直来照之，影则倒见。以手扪心而来，则见肠胃五脏，历然无碍。”

造句: 只除非天见怜，奈天天又远，今日个幸对清官，～。★元·关汉卿《望江亭》第四折

明来暗往 (míng lái àn wǎng)

缩写: mlaw

解释: 公开或暗地里来往。形容关系密切，往来频繁。

出自: 无

造句: 他们～地进行非法交易，胆大妄为。

明廉暗察 (míng lián àn chá)

缩写: mlac

解释: 犹言明察暗访。指用各种方法进行调查研究。

出自: 无

造句: 无

明媒正礼 (míng méi zhèng lǐ)

缩写: mmzl

解释: 犹言明媒正娶。旧指正式婚姻。

出自: 无

造句: 无

明媒正娶 (míng méi zhèng qǔ)

缩写: mmzq

解释: 明、正形容正大光明。旧指正式婚姻。

出自: 明·冯梦龙《古今小说》卷一论起初婚，王氏在前，只因休了一番，这平氏到是明媒正娶，又且平氏年长一岁，让平氏为正房，王氏反做偏房，两个姐妹相称。”

造句: 无

明明白白 (míng míng bái bái)

缩写: mmbb

解释: 清清楚楚，明确无误；正大光明。

出自: 明·施耐庵《水浒传》第45回账目已明明白白，并无分文来去。”

造句: 但须不设暗计，～，有手段平空拿我，吾即拜降。★《杨家将演义》第13回
