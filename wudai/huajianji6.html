🎉 四字成语6 🥳

论心定罪 (lùn xīn dìng zuì)

Abbreviation: lxdz

Explanation: 根据犯罪人的动机和情节来判定其罪行。

Derivation: 无

Example: 无

论议风生 (lùn yì fēng shēng)

Abbreviation: lyfs

Explanation: 谈论得极其生动而又风趣。

Derivation: 《宋史·陈亮传》亮为人才气超迈，喜谈兵；论议风生，下笔数千言立就。”

Example: 无

论资排辈 (lùn zī pái bèi)

Abbreviation: lzpb

Explanation: 论根据；资资历，资格；辈大小或前后顺序。根据资历深浅、辈份的大小决定级别、待遇的高低。

Derivation: 无

Example: 单位分房时论资排辈的现象不可避免。

捋虎须 (luō hǔ xū)

Abbreviation: lhx

Explanation: 捋抚摩。比喻触犯有权势的人或冒着很大的风险。

Derivation: 明·施耐庵《水浒传》第五回太公道‘好却甚好，只是不要捋虎须。’”

Example: 无

捋袖揎拳 (luō xiù xuān quán)

Abbreviation: lxxq

Explanation: 捋拉起。揎卷起袖子。拉起袖子，伸出拳头。形容怒气冲冲准备动武的样子。

Derivation: 元·金仁杰《追韩信》第四折前后军兵紧相并，左右枪刀厩围定。捋袖揎拳挺盔顶，破步撩衣扯剑迎。”

Example: 无

罗敷有夫 (luó fū yǒu fū)

Abbreviation: lfyf

Explanation: 旧指女子已有丈夫。

Derivation: 汉·无名氏《陌上桑》诗使君自有妇，罗敷自有夫。”

Example: 鸳鸯梦好两欢娱，记否罗敷自有夫。★清·纪昀《阅微草堂笔记》卷三

罗掘俱穷 (luó jué jù qióng)

Abbreviation: ljjq

Explanation: 罗雀掘鼠，而雀鼠已近；比喻无法筹到款项。

Derivation: 无

Example: 公司的财务危机已到～的地步，他却仍不轻言放弃。

罗掘一空 (luó jué yī kōng)

Abbreviation: ljyk

Explanation: 罗用网捕鸟；掘指挖掘老鼠洞找粮食。用尽一切办法，搜括财物殚尽。

Derivation: 蔡东藩、许廑父《民国通俗演义》第三十回是时宁城已罗掘一空，急切不得巨款，没奈何任他所为。”

Example: 无

罗钳吉网 (luó qián jí wǎng)

Abbreviation: lqjw

Explanation: 用以指酷虐诬陷。

Derivation: 《资治通鉴·唐玄宗天宝四载》李林甫欲除不附己者，重用酷吏罗希奭、吉温，二人皆随林甫所欲深浅，锻炼成狱，无能自脱者，时人谓之‘罗钳吉网’。”

Example: 堪伤！那些儿公道天开？端的是～。★明·沈鲸《双珠记·僧榻传音》

罗雀掘鼠 (luó què jué shǔ)

Abbreviation: lqjs

Explanation: 原指张网捉麻雀、挖洞捉老鼠来充饥的窘困情况，后比喻想眷法筹措财物。

Derivation: 《新唐书·张巡传》至罗雀掘鼠，煮铠弩以食。”

Example: 快饿死者，～，粮食罄尽，转为沟瘠是也。★严复《有如三保》

罗之一目 (luó zhī yī mù)

Abbreviation: lzym

Explanation: 网之一孔。比喻局部在整体中才能起其作用；脱离了整体，局部便不起作用。

Derivation: 《淮南子·说山训》有鸟将来，张罗而待之，得鸟者罗之一目也。今非昔比为一目之罗，则无时得鸟矣。”

Example: 一目之罗，不能得鸟，得鸟者～耳。★隋·智顗《摩诃止观》卷五

罗织构陷 (luò zhī gòu xiàn)

Abbreviation: lzgx

Explanation: 罗织收集编造；构陷用虚构的罪名去陷诬人。形容千方百计地陷害别人。

Derivation: 《后汉书·顺帝纪》王圣等惧有后祸，遂与丰京其构陷太子，太子坐废为济阴王。”

Example: 无

罗织罪名 (luó zhī zuì míng)

Abbreviation: lzzm

Explanation: 指捏造罪名，陷害无辜的人。

Derivation: 无

Example: 无

锣鼓喧天 (luó gǔ xuān tiān)

Abbreviation: lgxt

Explanation: 喧声音大。锣鼓震天响。原指作战时敲锣击鼓指挥进退◇多形容喜庆、欢乐的景象。

Derivation: 元·尚仲贤《单鞭夺槊》第四折早来到北邙前面，猛听的锣鼓喧天，那军不到三千，拥出个将一员，雄纠纠威风武艺显。”

Example: 周庸祐一一回发，赏封五块银子，各人称谢。少时，～，笙箫彻耳。★清·黄小配《廿载繁华梦》第二十二回

裸裎袒裼 (luǒ chéng tǎn xī)

Abbreviation: lctx

Explanation: 袒裼露臂；祼裎露体。指脱衣露体，没有礼貌。

Derivation: 《孟子·公孙丑上》尔为尔，我为我，虽袒裼裸裎于我侧，尔焉能浼我哉？”

Example: 无

裸袖揎拳 (luǒ xiù xuān quán)

Abbreviation: lxxq

Explanation: 挽起衣袖，露出拳头，形容浪荡。

Derivation: 元·张鸣善《水仙子·讥时》铺眉苫眼早三公，裸袖揎拳享万钟。”

Example: 无

攞袖揎拳 (luǒ xiù xuān quán)

Abbreviation: lxxq

Explanation: 挽起袖子，露出拳头，准备动武。

Derivation: 元·无名氏《碧桃花》第三折一个个气昂昂性儿不善，他每都叫吼吼攞袖揎拳。”

Example: 无

洛阳才子 (luò yáng cái zǐ)

Abbreviation: lycz

Explanation: 本指西汉贾谊。泛指有文学才华的人。

Derivation: 晋·潘岳《西征赋》终童山东之英妙，贾生洛阳之才子。”

Example: 洛阳城里春光好，～他乡老。★前蜀·韦庄《菩萨蛮》

洛阳纸贵 (luò yáng zhǐ guì)

Abbreviation: lyzg

Explanation: 比喻著作有价值，流传广。

Derivation: 《晋书·左思传》于是豪贵之家竞相传写，洛阳为之纸贵。”

Example: 人说～”，谁知今日闹到长安扇贵”。此时画的手也酸了，眼也花了。（清·李汝珍《镜花缘》第七十七回）

络绎不绝 (luò yì bù jué)

Abbreviation: lybj

Explanation: 形容行人车马来来往往，接连不断。

Derivation: 《后汉书·南匈奴传》窜逃入塞者络绎不绝。”

Example: 我梦里所到的地方，竟是一片康庄大道，马来车往，～。★清·李宝嘉《官场现形记》第六十回
