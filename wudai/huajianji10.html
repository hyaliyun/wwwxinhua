一反常态 (yī fǎn cháng tài)

Abbreviation: yfct

Explanation: 完全改变了平时的态度。

Derivation: 郭澄清《大刀记》第十六章可是而今，他却一反常态，神采飞扬地高谈阔论起来。”

Example: 他平时总是有说有笑，今天却～，默不作声。

一反既往 (yī fǎn jì wǎng)

Abbreviation: yfjw

Explanation: 一完全；既往从前。完全与以前相反。

Derivation: 无

Example: 从今以后，他要一反既往。

一反其道 (yī fǎn qí dào)

Abbreviation: yfqd

Explanation: 一整个，完全；反相反；其他的；道轨道。完全与以前的轨道相反。

Derivation: 无

Example: 他总是喜欢一反其道而行之。

一饭千金 (yī fàn qiān jīn)

Abbreviation: yfqj

Explanation: 比喻厚厚地报答对自己有恩的人。

Derivation: 《史记·淮阴侯列传》信钓于城下，诸漂母漂，有一母见信饥，饭信，竟漂数十日。”又信至国，如所从食漂母，赐千金。”

Example: 马周道壁上诗句犹在，～岂可忘也。”★明·凌濛初《二刻拍案惊奇》卷五

一饭三吐哺 (yī fàn sān tǔ bǔ)

Abbreviation: yfstb

Explanation: 指一顿饭之间，三次停食，以接待宾客。比喻求贤殷切。

Derivation: 《史记·鲁周公世家》我文王之子，武王之弟，成王之叔父，我于天下亦不贱矣。然我一沐三握发，一饭三吐哺，起以待士，犹恐失天下之贤人。子之鲁，慎勿以国骄人。”

Example: 干王恩高量广，不以初交见嫌，不以触犯见罪，虽与周之姬旦，～，一沐三握发，无以过也。★太平天国·洪仁玕《英杰归真》

一饭之德 (yī fàn zhī dé)

Abbreviation: yfzd

Explanation: 比喻微小的恩德。

Derivation: 西汉·司马迁《史记·范雎蔡泽列传》范雎于是散家财物，尽以报所尝困厄者。一饭之德必偿，睚眦之怨必报。”

Example: 世相传余初有～，葛如方布衣野走。★清·俞樾《茶香室续钞·陆晋》

一方之任 (yī fāng zhī rèn)

Abbreviation: yfzr

Explanation: 负一方面责任的职务。

Derivation: 《汉书·终军传》不足以亢一方之任，窃不胜愤懑。”

Example: 无

一飞冲天 (yī fēi chōng tiān)

Abbreviation: yfct

Explanation: 鸟儿展翅一飞，直冲云霄。比喻平时没有特殊表现，一下做出了惊人的成绩。

Derivation: 《韩非子·喻老》虽无飞，飞必冲天；虽无鸣，鸣必惊人。”《史记·滑稽列传》此鸟不飞则已，一飞冲天；不鸣则已，一鸣惊人。”

Example: 无

一吠百声 (yī fèi bǎi shēng)

Abbreviation: yfbs

Explanation: 即一犬吠形，百犬吠声。

Derivation: 明·何景明《上杨邃庵书》群猜共怒，一吠百声，持辩风起，发言雷同矣。”

Example: 无

一分耕耘，一分收获 (yī fēn gēng yún，yī fēn shōu huò)

Abbreviation: yfgyfsh

Explanation: 付出一份劳力就得一分收益。

Derivation: 无

Example: ～，你多做一些事，只有更多收益，绝不会吃亏的。

一分为二 (yī fēn wéi èr)

Abbreviation: yfwe

Explanation: 哲学用语，指事物作为矛盾的统一体，都包含着相互矛盾对立的两个方面。通常指全面看待人或事物，看到积极方面，也看到消极方面。

Derivation: 无

Example: 无

一分一毫 (yī fēn yī háo)

Abbreviation: yfyh

Explanation: 形容很少的数量。

Derivation: 朱自清《给亡妇》那里有一分一毫想着你自己。”

Example: 我们不能拿群众的～。

一风吹 (yī fēng chuī)

Abbreviation: yfc

Explanation: 比喻完全勾销（多指决定、结论等）。

Derivation: 姚雪垠《李自成》第一卷第十七章后世不论官宦和平民人家，只要是有产业的，兄弟叔侄争产，势同仇人，平日所讲的仁义忠信，兄友弟恭，全都一风吹了。”

Example: 无

一佛出世 (yī fó chū shì)

Abbreviation: yfcs

Explanation: 犹一佛出世，二佛生天。

Derivation: 明·施耐庵《水浒传》第九回［差拨］把林冲骂得一佛出世，那里敢抬头应答。”

Example: 遂把李三上起刑法来，打得～，二佛生天，只不肯招。★明·凌濛初《二刻拍案惊奇》卷三十八

一佛出世，二佛涅盘 (yī fó chū shì，èr fó niè pán)

Abbreviation: yfcsfnp

Explanation: 佛家称生为出世，死为涅盘。指死去活来。

Derivation: 明·施耐庵《水浒全传》第三十九回打得宋江一佛出世，二佛涅盘，皮开肉绽，鲜血淋漓。”

Example: 无

一佛出世，二佛涅槃 (yī fó chū shì，èr fó niè pán)

Abbreviation: yfcsfnp

Explanation: 佛家称生为出世”，死为涅槃”。指死去活来。

Derivation: 明·施耐庵《水浒全传》第三十九回打得宋江一佛出世，二佛涅槃，皮开肉绽，鲜血淋漓。”

Example: 无

一佛出世，二佛升天 (yī fó chū shì，èr fó shēng tiān)

Abbreviation: yfcsfst

Explanation: 死去活来之意。出世，生；生天，死。

Derivation: 明·冯梦龙《醒世恒言》第三十八卷［李清］直等到一佛出世，二佛升天，方才有个青衣童子开门出来。”

Example: 陈月波、马剑郎只好把气出在那两个卫兵的身上，骂得他们～。★侯枫《海陆丰农民运动的领导者彭湃》

一佛出世，二佛生天 (yī fó chū shì，èr fó shēng tiān)

Abbreviation: yfcsfst

Explanation: 死去活来之意。出世，生；生天，死。

Derivation: 明·凌濛初《二刻拍案惊奇》卷二十三合家见防御说得伤心，一齐号哭起来。直哭得一佛出世，二佛生天，连崔生也不知陪下了多少眼泪。”

Example: 无

一夫当关 (yī fū dāng guān)

Abbreviation: yfdg

Explanation: 形容地势十分险要，易守难攻。

Derivation: 《明史·外国传一·朝鲜》忠州左右乌、竹两岭，羊肠绕曲，有‘一夫当关’之险。”

Example: 无

一夫当关，万夫莫开 (yī fū dāng guān，wàn fū mò kāi)

Abbreviation: yfdgfmk

Explanation: 意思是山势又高又险，一个人把着关口，一万个人也打不进来。形容地势十分险要。

Derivation: 《淮南子·兵略训》一人守隘，而千人弗敢过也。”晋·左思《蜀都赋》一人守隘，万夫莫向。”

Example: 剑阁峥嵘而崔嵬，～。★唐·李白《蜀道难》诗
