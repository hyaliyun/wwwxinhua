随事制宜 (suí shì zhì yí)

缩写: sszy

解释: 根据事实，采取适当措施。

出自: 宋·朱熹《朱子语类》卷三十七如道之正体，却一定于此，而随事制宜，自莫不当。”

造句: 无

随俗沉浮 (suí sú chén fú)

缩写: sscf

解释: 顺随世俗，没有己见。

出自: 《晋书·文苑传·王沈》少有俊才，出于寒素，不能随俗沉浮，为时豪所抑。”

造句: 无

随俗浮沉 (suí sú fú chén)

缩写: ssfc

解释: 自己没有一定的想法，随着潮流走。

出自: 汉·司马迁《报任少卿书》故且从俗浮沉，与时俯仰，以通其狂惑。”

造句: 无

随物赋形 (suí wù fù xíng)

缩写: swfx

解释: 指针对客观事物本身的不同形态给予形象生动的描绘。

出自: 无

造句: 无

随物应机 (suí wù yīng jī)

缩写: swyj

解释: 顺应事物而机变。

出自: 无

造句: 无

随乡入俗 (suí xiāng rù sú)

缩写: sxrs

解释: 到一个地方就按照这一个地方的风俗习惯生活。也比喻到什么地方都能适应。同随乡入乡”。

出自: 无

造句: 无

随乡入乡 (suí xiāng rù xiāng)

缩写: sxrx

解释: 到一个地方就按照这一个地方的风俗习惯生活。也比喻到什么地方都能适应。

出自: 《庄子·山木》入其俗，从其令。”宋·范成大《秋雨快晴静胜堂席上》诗天涯节物遮愁眼，旦复随乡便入乡。”

造句: ～”，到了你这里，自然把这金珠玉宝一概贬为俗器了。★清·曹雪芹《红楼梦》第四十一回

随心所欲 (suí xīn suǒ yù)

缩写: sxsy

解释: 随任凭；欲想要。随着自己的意思，想要干什么就干什么。

出自: 《论语·为政》七十而从心所欲，不逾矩。”

造句: 我们二人并坐，～的漫谈。★臧克家《老舍永在》

随行就市 (suí háng jiù shì)

缩写: shjs

解释: 指按市场行情行事。

出自: 无

造句: 一般农副产品的购销，可～，灵活经营。

随行逐队 (suí xíng zhú duì)

缩写: sxzd

解释: 指跟着大家一道行动。

出自: 明·冯梦龙《喻世明言》第十八卷众倭公然登岸，少不得放火杀人，杨八老虽然心中不愿，也不免随行逐队。”

造句: 无

随寓而安 (suí yù ér ān)

缩写: syea

解释: 指能顺应环境，在任何境遇中都能满足。同随遇而安”。

出自: 无

造句: 无

随寓随安 (suí yù suí ān)

缩写: sysa

解释: 犹言随遇而安。指能顺应环境，在任何境遇中都能满足。

出自: 无

造句: 无

随遇而安 (suí yù ér ān)

缩写: syea

解释: 随顺从；遇遭遇。指能顺应环境，在任何境遇中都能满足。

出自: 清·刘献廷《广阳杂记》一随寓而安，斯真隐矣。”

造句: 不过能够～——即有船坐船云云——则比起幻想太多的人们来，可以稍为安稳，能够敷衍下去而已。★鲁迅《两地书》六

随圆就方 (suí yuán jiù fāng)

缩写: syjf

解释: 处事顺应形势和情况的变化；待人随和而不固执。同随方就圆”。

出自: 无

造句: 无

随缘乐助 (suí yuán lè zhù)

缩写: sylz

解释: 随着缘分的深浅，乐意捐助多少就捐助多少。

出自: 清·黄小配《廿载繁华梦》第二十九回这都是随缘乐助，本不能强人的，或多或少，却是未定，总求大人这里踊跃些便是。”

造句: 无

随踵而至 (suí zhǒng ér zhì)

缩写: szez

解释: 踵脚后跟。一个跟着一个前来。形容人来得多，接连不断。

出自: 《战国策·齐策三》子来，寡人闻之，千里而一士，是比肩而立，百世而一圣，若随踵而至也，今子一朝而见七士，则士不亦众乎！”

造句: 无

随珠弹雀 (suí zhū tán què)

缩写: sztq

解释: 用夜明珠去弹鸟雀。比喻做事不知道衡量轻重，因而得到的补偿不了失去的。

出自: 《庄子·让王》今且有人于此，以随侯之珠，弹千仞之雀，世必笑之。是何也？以其所用者重，而所要者轻也。”

造句: 无

随珠和璧 (suí zhū hé bì)

缩写: szhb

解释: 随侯珠与和氏璧的并称。泛指珍宝或珍宝中的上品。事见《韩非子·和氏》、《淮南子·览冥训》。

出自: 传说古代随国姬姓诸侯见一大蛇伤断，以药敷之而愈；后蛇于江中衔明月珠以报德，因曰随侯珠，又称灵蛇珠。楚人卞和于荆山得一璞玉，先后献给武王、文王，均以为石，和以欺君罪被砍断两足；成王登位，使人剖璞，果得夜...

造句: 兴造早乙之帐，落以～。★《汉书·西域传赞》

随珠荆玉 (suí zhū jīng yù)

缩写: szjy

解释: 随侯珠与和氏璧的并称。泛指珍宝或珍宝中的上品。同随珠和璧”。

出自: 无

造句: 无

岁比不登 (suì bǐ bù dēng)

缩写: sbbd

解释: 比屡屡；频频。农业连年歉收。同比年不登”。

出自: 《汉书·严助传》数年岁比不登，民待卖爵赘子以接衣食。”

造句: 壬寅，以诸路～，免今年田租、丝银。★《元史·世祖纪七》
