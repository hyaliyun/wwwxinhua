仙姿玉色 (xiān zī yù sè)

Abbreviation: xzys

Explanation: 形容女子姿态容貌极美。

Derivation: 明·谢谠《四喜记·巧夕宫筵》宫中郑娘娘，乃是郑参政之女，数月前选入宫中，仙姿玉色，世上无双。”

Example: 无

仙姿玉质 (xiān zī yù zhì)

Abbreviation: xzyz

Explanation: 形容姿容清秀品格高洁。

Derivation: 无

Example: 无

先得我心 (xiān dé wǒ xīn)

Abbreviation: xdwx

Explanation: 得适合，契合。比喻别人先说出了自己心里想说的话。

Derivation: 《孟子·告子上》心之所同然者何也？谓理也、义也。圣人先得我心之所同然耳，故理义之悦我心，犹刍豢之悦我口。”

Example: 心之所同然者何也？谓理也，义也，性善也。圣人～之所同然耳，性相近也。★清·陈澧《东塾读书记·孟子》

先睹为快 (xiān dǔ wéi kuài)

Abbreviation: xdwk

Explanation: 睹看见。以能尽先看到为快乐。形容盼望殷切。

Derivation: 唐·韩愈《与少室李拾遗书》朝廷之士，引颈东望，若景星凤皇之始见也，争先睹之为快。”

Example: 现在我们特地和西滢先生商量，把闲话”选集起来，印成专书，留心文艺界掌故的人，想必都以～。★鲁迅《而已集·革首领”》

先断后闻 (xiān duàn hòu wén)

Abbreviation: xdhw

Explanation: 指先处理后奏闻。

Derivation: 无

Example: 无

先发制人 (xiān fā zhì rén)

Abbreviation: xfzr

Explanation: 发开始行动；制控制，制服。原指丢动手的牌主动地位，可以控制对方◇也泛指争取主动，先动手来制服对方。

Derivation: 《汉书·项籍传》先发制人，后发制于人。”

Example: 剑波没等这位警长开口，便理直气壮地来个～。★曲波《林海雪原》三

先公后私 (xiān gōng hòu sī)

Abbreviation: xghs

Explanation: 先以公事为重，然后考虑私事。

Derivation: 《三国志·魏志·杜恕传》忧公忘私者必不然，但先公后私即自办也。”

Example: 卿～，可谓大忠矣。★明·冯梦龙《东周列国志》第九十七回

先号后庆 (xiān hào hòu qìng)

Abbreviation: xhhq

Explanation: 指初凶后吉。

Derivation: 无

Example: 无

先号后笑 (xiān hào hòu xiào)

Abbreviation: xhhx

Explanation: ①指命运先凶后吉。②指文章或著作结尾比开头精彩。

先河后海 (xiān hé hòu hǎi)

Abbreviation: xhhh

Explanation: 先祭河神，后祭海神。比喻治学要弄清源流。

Derivation: 《礼记·学记》三王之祭川也，皆先河而后海，或源也，或委也，此之谓务本。”

Example: 无

先花后果 (xiān huā hòu guǒ)

Abbreviation: xhhg

Explanation: 旧时比喻先生女后生男。

Derivation: 明·冯梦龙《醒世恒言》第二十七卷生下三女一男儿子名曰承祖，长女名玉英，次女名桃英，三女名月英。元来是先花后果的。倒是玉英居长，次即承祖。”

Example: 无

先见之明 (xiān jiàn zhī míng)

Abbreviation: xjzm

Explanation: 明·指眼力。事先看清问题的能力。指对事物发展的预见性。

Derivation: 《后汉书·杨彪传》愧无日磾先见之明，犹怀老牛舐犊之爱。”

Example: 他觉得胆小的杜竹斋有时候实在颇具～，因而也省了多少烦恼。★茅盾《子夜》十七

先来后到 (xiān lái hòu dào)

Abbreviation: xlhd

Explanation: 按照来到的先后确定次序。

Derivation: 明·无名氏《南牢记》第二折也有个先来后到，反教俺无下梢。”

Example: 无

先礼后兵 (xiān lǐ hòu bīng)

Abbreviation: xlhb

Explanation: 礼礼貌；兵武力。先按通常的礼节同对方交涉，如果行不能，再用武力或其他强硬手段解决。

Derivation: 明·罗贯中《三国演义》第十一回刘备远来救援，先礼后兵，主公当用好言答之，以慢备心，然后进兵攻城，城可破也。”

Example: 你这～方法实在高明。

先难后获 (xiān nán hòu huò)

Abbreviation: xnhh

Explanation: 难艰难，劳苦；获收获。先付出劳动然后再取得收获。比喻不坐享其成。

Derivation: 《论语·雍也》仁者，先难而后获，可谓仁矣。”

Example: 无

先驱蝼蚁 (xiān qū lóu yǐ)

Abbreviation: xqly

Explanation: 蝼蚁蝼蛄和蚂蚁。自己先死埋葬地下，为别人驱除蝼蚁。比喻效命于人，不惜先死。

Derivation: 《战国策·楚策》臣入则编席，出则陪乘。大王万岁千秋之后，愿得以身试黄泉，蓐蝼蚁。”

Example: 人以寿为欢，老臣以寿为戚。既不能～，乃复频见此事。★明·李贽《初潭集·君臣二》

先人后己 (xiān rén hòu jǐ)

Abbreviation: xrhj

Explanation: 首先考虑别人，然后想到自己。

Derivation: 《礼记·坊记》子云‘君子贵人而贱己，先人而后己。’”

Example: 这也表现了他～、艰苦朴素的崇高品质。★朱敏《深情怀念周伯伯》

先入为主 (xiān rù wéi zhǔ)

Abbreviation: xrwz

Explanation: 指先听进去的话或先获得的印象往往在头脑中占有主导地位，以后再遇到不同的意见时，就不容易接受。

Derivation: 《汉书·息夫躬传》唯陛下观览古今，反复参考，无以先入之语为主。”

Example: 见长幼尊卑之节有一定不易之理，～，故后起之私心，终有所顾忌而不敢逞。★清·阮葵生《茶余客话》卷七

先入之见 (xiān rù zhī jiàn)

Abbreviation: xrzj

Explanation: 指在对某事物进行调查研究之前就形成或接受的看法。指成见。

Derivation: 《汉书·息夫躬传》唯陛下观览古今，反复参考，无以先入之语为主。”

Example: 无

先声夺人 (xiān shēng duó rén)

Abbreviation: xsdr

Explanation: 先张扬自己的声势以压倒对方。也比喻做事抢先一步。

Derivation: 《左传·宣公十二年》军志曰‘先人有夺人之心，’薄之也。”

Example: 他深深懂得～的道理，就预先做了广告，因此货销售得很快。
