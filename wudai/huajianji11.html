援古刺今 (yuán gǔ cì jīn)

Abbreviation: ygcj

Explanation: 引述故事古义以讽刺今事。

Derivation: 明·杨慎《升庵诗话》卷六援古刺今，箴戒得失，谓之箴。”《天讨·豕韦之裔〈普告汉人〉》述怀咏史，不过词人之习态，不可以为援古刺今。”

Example: 无

援古证今 (yá gǔ zhèng jīn)

Abbreviation: ygzj

Explanation: 援引。引述古事来证明今事。

Derivation: 南朝·梁·刘勰《文心雕龙·事类》事类者，盖文章之外，据事以类义，援古以证今者也。”

Example: ～，极力为之。★清·黄宗羲《柳敬亭传》

缘木求鱼 (yuán mù qiú yú)

Abbreviation: ymqy

Explanation: 缘木爬树。爬到树上去找鱼。比喻方向或办法不对头，不可能达到目的。

Derivation: 《孟子·梁惠王上》以若所为，求若所欲，犹缘木而求鱼也。”

Example: 若想善出此关，大王乃～，非徒无益，而又害之也。★明·许仲琳《封神演义》第三十三回

缘悭命蹇 (yuán qiān mìng jiǎn)

Abbreviation: yqmj

Explanation: 缘缘分。悭吝俭，欠缺。蹇不顺利。缘分浅薄，命运不好，

Derivation: 无

Example: 无

缘情体物 (yuán qíng tǐ wù)

Abbreviation: yqtw

Explanation: 缘因；体描写。抒发感情，描写事物。

Derivation: 晋·陆机《文赋》诗缘情而绮靡，赋体物而浏亮。”

Example: 僧问举子入门时为何？”师曰～事作么生？”★宋·释普济《五灯会元》卷十四

缘文生义 (yuán wén shēng yì)

Abbreviation: ywsy

Explanation: 文文字，指字面。只根据字面理解内容，牵强附会，不求甚解。亦作望文生义”。

Derivation: 无

Example: 无

鼋鸣鳖应 (yuán míng biē yìng)

Abbreviation: ymby

Explanation: 比喻互相感应，一倡一和。

Derivation: 《后汉书·张衡传》高祖踞洗，以对郦生，当此之会，乃鼋鸣而鳖应也，故能同心戮力。”

Example: 无

源清流洁 (yuán qīng liú jié)

Abbreviation: yqlj

Explanation: 源头的水清，下游的水也清。原比喻身居高位的人好，在下面的人也好。也比喻事物的因果关系。

Derivation: 《荀子·君道》源清则流清，源浊则流浊。”

Example: ～，本盛末荣。★汉·班固《泗水亭碑铭》

源清流清 (yuán qīng liú qīng)

Abbreviation: yqlq

Explanation: 源头的水清，下游的水自然就清。比喻因果相连，事物的本原好，其发展和结局也就好；或领导贤明，其下属也廉洁。源，也作原。亦作源清流洁”、源清流净”。

Derivation: 无

Example: 无

源泉万斛 (yuán quán wàn hú)

Abbreviation: yqwh

Explanation: 比喻文思涌溢。

Derivation: 宋·苏轼《文说》吾文如万斛泉源，不择地而出。”

Example: 无

源头活水 (yuán tóu huó shuǐ)

Abbreviation: yths

Explanation: 原比喻读书越多，道理越明。现也指事物发展的动力和源泉。

Derivation: 宋·朱熹《观书有感》诗问渠哪得清如许，谓有源头活水来。”

Example: 无

源源本本 (yuán yuán běn běn)

Abbreviation: yybb

Explanation: 源头和根本。多用以指事情的始末。

Derivation: 清·李宝嘉《文明小史》第六十回自从这立宪二字发见了，就有人从西书上译出一部宪法新论，讲的源源本本，有条有理。”

Example: 我已经查明白～是怎么一回事。★茅盾《子夜》十五

源源不断 (yuán yuán bù duàn)

Abbreviation: yybd

Explanation: 形容接连不断。

Derivation: 《孟子·万章上》欲常常而见之，故源源而来。”

Example: 夜间，官军打通了由智亭山通往龙驹寨的大道，所以从天亮起就有军粮～地从龙驹寨向西运送。（姚雪垠《李自成》第二卷第十三章）

源源不绝 (yuán yuán bù jué)

Abbreviation: yybj

Explanation: 源源水流不断的样子。形容接连不断。

Derivation: 《孟子·万章上》欲常常而见之，故源源而来。”

Example: 无

源源而来 (yuán yuán ér lái)

Abbreviation: yyel

Explanation: 原指诸侯相继朝觐一辈子◇形容接连不断地到来。

Derivation: 《孟子·万章上》欲常常而见之，故源源而来。”

Example: 果然有了此好，方士～，零零星星，也弄去了好些银子，受过了好些丹客的哄骗。★明·凌濛初《初刻拍案惊奇》卷十八

源远流长 (yuán yuǎn liú cháng)

Abbreviation: yylc

Explanation: 源头很远，水流很长。比喻历史悠久。

Derivation: 唐·白居易《海州刺史裴君夫人李氏墓志铭》夫源远者流长，根深者枝茂。”

Example: 剥削阶级的幸福观，是～，影响仍然是相当深广的。★秦牧《一个黑人战士的豪迈声音》

猿悲鹤怨 (yuán bēi hè yuàn)

Abbreviation: ybhy

Explanation: 猿和鹤凄厉地啼叫。

Derivation: 宋·朱熹《送籍溪胡丈赴馆供职》诗之二猿悲鹤怨因何事？只恐先生袖手归。”

Example: 无

猿鹤虫沙 (yuán hè chóng shā)

Abbreviation: yhcs

Explanation: 旧时比喻战死的将士。也指死于战乱的人。

Derivation: 《太平御览》卷九六一引《抱朴子》周穆王南征，一军尽化，君子为猿为鹤，小人为虫为沙。”唐·韩愈《送区弘南归诗》穆昔南征军不归，虫沙猿鹤伏似飞。”

Example: 无

猿鹤沙虫 (yuán hè shā chóng)

Abbreviation: yhsc

Explanation: 指阵亡的将士或死于战乱的人民。

Derivation: 《艺文类聚》卷九十引晋葛洪《抱朴子》周穆王南征，一军尽化，君子为猿为鹤，小人为虫为沙。”按，今本《抱朴子·释滞》作山徙社移，三军之众，一朝尽化，君子为鹤，小人成沙。”

Example: 你看山川图画，人物衣冠，生聚年时久，忍下得咸阳一炬里变焦丘，～满地愁。★清·杨潮观《下江南曹彬誓众》

猿猴取月 (yuán hóu qǔ yuè)

Abbreviation: yhqy

Explanation: 比喻愚昧无知。也比喻白费力气。

Derivation: 宋·黄庭坚《沁园春》词镜里拈花，水中捉月，觑着无由得近伊。”

Example: 无
